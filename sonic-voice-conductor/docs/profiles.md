# Profile

## warmup
- BPM: 115–128, Default 122
- Energie: 0.2–0.6
- Geeignet für Set-Start

## peak
- BPM: 125–135, Default 128
- Energie: 0.5–1.0
- Alle Aktionen erlaubt

## afterhour
- BPM: 110–122, Default 118
- Dunkelheit: 0.5–0.95
- Ruhigere Übergänge

## industrial
- BPM: 130–145, Default 135
- Energie: 0.6–1.0
- Härtere Grenzen
